# Typescript Workshop

First challenge is to convert Cat Fact! app to typescript

## Setup

1. Install [Node.js](https://nodejs.org) (see `.nvmrc` for required verion).
2. Install [Yarn](https://yarnpkg.com).

## Installation

`yarn install`

## Development

To start webpack dev server run:

`yarn start`
